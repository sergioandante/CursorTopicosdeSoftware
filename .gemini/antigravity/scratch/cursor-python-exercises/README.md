# Ejercicios de Python con Cursor

Este repositorio contiene las soluciones a los ejercicios propuestos en el curso de "Cursor con Python".

## Ejercicios Incluidos

1. **Calculadora Simple (`calculadora.py`)**:
   - Realiza operaciones básicas (suma, resta, multiplicación, división).
   - Maneja errores como la división por cero.
   - Funciona en un ciclo continuo hasta que el usuario escribe "salir".

2. **FizzBuzz (`fizzbuzz.py`)**:
   - Recorre números del 1 al 50.
   - Aplica las reglas clásicas de FizzBuzz.

3. **Análisis de Datos Básico (`analisis.py`)**:
   - Lee un archivo CSV (`data.csv`).
   - Calcula media, mediana y desviación estándar.
   - Genera una gráfica de dispersión usando `matplotlib`.

## Cómo ejecutar

Asegúrate de tener instaladas las dependencias necesarias para el ejercicio de análisis:

```bash
pip install pandas matplotlib
```

Luego puedes ejecutar cualquier script:

```bash
python calculadora.py
python fizzbuzz.py
python analisis.py
```
